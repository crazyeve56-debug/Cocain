# 데이터 사이언티스트 포트폴리오 웹사이트

이 저장소는 간단한 데이터 사이언티스트 포트폴리오 정적 사이트 템플릿을 포함합니다. 한국어로 된 기본 섹션(소개, 프로젝트, 이력서, 연락)을 제공합니다.

로컬에서 확인하는 방법

1. 파일을 브라우저로 바로 열기

```bash
# index.html 파일을 브라우저로 열면 동작합니다.
xdg-open index.html
```

2. 간단한 로컬 서버 사용 (권장)

```bash
# 파이썬 3이 설치되어 있을 때
python3 -m http.server 8000
# 그런 다음 브라우저에서 http://localhost:8000 을 엽니다.
```

배포

- GitHub Pages: 이 저장소의 `main` 브랜치를 Pages로 설정하거나, `gh-pages` 브랜치로 빌드하여 배포할 수 있습니다.
- Netlify, Vercel 등 정적 호스팅 서비스에 업로드하면 됩니다.

커스터마이징 가이드

- `index.html`에서 이름, 소개, 이메일, 프로젝트 링크 등을 편집하세요.
- 프로젝트 섹션에 실제 저장소 링크와 데모 링크를 추가하세요.
- `styles.css`에서 색상과 폰트를 수정해 개인 브랜딩에 맞춥니다.
